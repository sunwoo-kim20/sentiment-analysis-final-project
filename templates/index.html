The Process

This project aims to create a neural network that can predict the sentiment of tweets from Twitter.
Want to jump straight to the code? GitHub Repository

The Model

The model is trained and tested on an IMDB movie reviews dataset with prescribed positive or negative rankings.
Click on the code snippet to see the full model in GitHub.

Creating SQL Tables

A SQL database and tables were then created to house our future Twitter data. This files runs automatically when the Flask app is run.
Click on the code snippet to see the full database tables creation in GitHub.

The API Call

Once the model was trained on the IMDB data, we were ready to start using it on other types of data. We used Twitter's API to call 10 tweets at a time. These tweets are then inserted into our SQL database for future use.
Click on the code snippet to see the full API call and database load in GitHub.

Serving it Up

The model and api call come together in a Flask app.
Click on the code snippet to see the full Flask App in GitHub.

Getting Interactive

A tweet is loaded from the database to our Sentiment Voting page. From here, users have the ability to vote on the tweet as well. These votes are stored in the database for future model retraining and analysis.
Click on the code snippet to see the full JavaScript in GitHub.

Feeding The Model

When a tweet is loaded to the voting page, it is cleaned and fed to the model to predict a sentiment level between 0-1. Our model's prediction can be seen on the voting page.
Click on the code snippet to see the full predict model in GitHub.

Looking Ahead

As more tweets are voted on, our cunfusion matrix is updated. This allows us to see how our algorithm is performing.
Click on the matrix to see the code in GitHub.

We Need You!

Want to be part of the action? We need your help to continue training the model. It's simple:

Step 1: Head on over to our Sentiment Voting page
Step 2: Read the tweet displayed at the top of the page
Step 3: Click the frown face for negative or the smile face for positive
Step 4: Have fun! Vote as many times as you would like.

Having a hard time deciding? That's okay! These close call tweets can go either way. Just vote how you feel.
If it is truly a neutral post, or does not contain enough actual text to illicit a sentiment, feel free to skip to the next tweet.
